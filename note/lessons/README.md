# 授業記録テンプレート 使い方ガイド

## 📋 テンプレート構造

このテンプレートは、以下の3つのタブで構成されます：

### 1. 📝 復習クイズ

前回の授業内容を確認するクイズです。

- 「前回の復習」セクションの「復習クイズ」に記載
- カルーセル形式で1問ずつ表示
- 選択肢をクリックすると即座に答え合わせ

### 2. 📅 本日のおさらい

今日の授業の概要と資料をまとめます。

- 前回のおさらい（あらすじ）
- 今日の流れ（学習項目リスト）
- 資料（スライドや配布資料へのリンク）

### 3. 📋 まとめクイズ

今日の授業内容を確認するクイズです。

- 「まとめクイズ」セクションに記載
- カルーセル形式で1問ずつ表示
- 選択肢をクリックすると即座に答え合わせ

## ✍️ 記入方法

### フロントマター

```yaml
---
date: 2026-02-01          # 授業日（YYYY-MM-DD形式）
subject: プログラミング    # 科目名
period: 5                 # 時限
unit: 変数とデータ型の基礎 # 単元名
---
```

### 前回の復習

```markdown
## 📚 前回の復習

**あらすじ**: 前回の授業内容を1-2文で簡潔にまとめます。

### 復習クイズ

**Q1**: 問題文を記入

- A) 選択肢1 ✓
- B) 選択肢2
- C) 選択肢3
```

**重要**: 正解の選択肢には `✓` を付けてください。

### 本日の予定

```markdown
## 📅 本日の予定

1. 学習項目1
2. 学習項目2
3. 学習項目3
```

### 資料

```markdown
## 📎 資料

- [スライド](https://docs.google.com/presentation/d/example)
- [配布資料](https://example.com/material.pdf)
```

**対応形式**:

- PDFファイル: 自動的に埋め込みビューアで表示
- Google Drive: プレビュー形式で表示
- その他のリンク: 新しいタブで開くリンクとして表示

### まとめクイズ

```markdown
## ✅ まとめクイズ

**Q1**: 問題文を記入

- A) 選択肢1 ✓
- B) 選択肢2
- C) 選択肢3
```

## 🎯 クイズ作成のポイント

1. **問題番号**: Q1, Q2, Q3... と連番で記載
2. **選択肢**: A), B), C) の形式で記載
3. **正解マーク**: 正解の選択肢の末尾に `✓` を付ける
4. **空行**: 問題文と選択肢の間に空行を入れる

### 良い例

```markdown
**Q1**: プログラミングとは何ですか？

- A) コンピュータに命令を書くこと ✓
- B) パソコンを修理すること
- C) インターネットを使うこと
```

### 悪い例（空行なし）

```markdown
**Q1**: プログラミングとは何ですか？
- A) コンピュータに命令を書くこと ✓
- B) パソコンを修理すること
```

## 🚀 変換方法

1. `note/lessons/` フォルダにマークダウンファイルを保存
2. ターミナルで以下のコマンドを実行:

   ```
   node note/scripts/convert.js
   ```

3. `note/[日付]-[科目名]/index.html` が生成されます

## 💡 Tips

- 復習クイズは2-3問程度が適切
- まとめクイズは3-5問程度が適切
- 問題文は簡潔に、選択肢は明確に
- 資料は授業前に準備してリンクを記載
