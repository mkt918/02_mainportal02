## 📅 本日の内容

### 1. なぜコンピュータは「2進数」を使うのか？

私たちが普段使っている数字は「10進数（0〜9の10個の数字を使う）」ですが、コンピュータの内部では電気の「ON」と「OFF」（または電圧の高低）で情報を処理しています。
そのため、**「0」と「1」の2種類の数字だけを使う「2進数」**のほうが、コンピュータの構造にとって非常に都合が良いのです。

- **0** → 電気が流れていない（OFF）
- **1** → 電気が流れている（ON）

この「0か1か」の状態一つ分が、前回学習した情報の最小単位である「**ビット (bit)**」です。

### 2. 2進数のしくみと数え方

10進数では、1つの桁が「9」の次になると繰り上がって「10」になります。
2進数では、使える数字が0と1だけなので、「1」の次ですぐに繰り上がりが発生し「10（イチゼロ）」となります。

| 10進数 | 2進数 |
| :---: | :---: |
| 0 | 0 |
| 1 | 1 |
| 2 | 10 |
| 3 | 11 |
| 4 | 100 |
| 5 | 101 |

### 3. 基数変換（10進数 → 2進数）

10進数を2進数に変換するには、「**2で割り算を続け、その余りを下から上に並べる**」という方法（スダレ算）が便利です。

**例：10進数の「13」を2進数にする**

1. 13 ÷ 2 = 6 あまり **1** （←これが1の位）
2. 6 ÷ 2 = 3 あまり **0** （←これが2の位）
3. 3 ÷ 2 = 1 あまり **1** （←これが4の位）
4. 1 ÷ 2 = 0 あまり **1** （←これが8の位）

下から読んで、答えは **`1101`** になります。これを**(13)₁₀ = (1101)₂** のように表記することがあります。

### 4. 基数変換（2進数 → 10進数）

2進数を10進数に変換するには、「**各桁の重み（位取り）を足し合わせる**」方法を使います。

2進数の桁の重みは、右から順番に次のように大きくなります。
`1の位` → `2の位` → `4の位` → `8の位` → `16の位` ...

**例：2進数の「1011」を10進数にする**
「1」が立っている桁の重みを足し算します。

- 8の位が「1」
- 4の位が「0」
- 2の位が「1」
- 1の位が「1」

**式**: 8 + 0 + 2 + 1 = **11**
よって、答えは **11** になります。
